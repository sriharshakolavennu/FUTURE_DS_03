STUDENT FEEDBACK & SATISFACTION ANALYSIS – POWER BI DASHBOARD

Project Overview  
This project analyzes student feedback and satisfaction data using a Power BI dashboard.  
The dashboard provides insights into overall student ratings, feedback count, sentiment distribution, course recommendation scores, and satisfaction levels.  
It helps understand student experience, identify improvement areas, and support data-driven decisions for enhancing course quality.

Dataset Information  
● Source: Student Feedback Survey Dataset  
● Format: CSV  
● Records: Student feedback responses  
● Granularity: Student-level feedback and ratings  

Key Columns Used  
● Overall Rating – Student overall course rating  
● Feedback Count – Total number of feedback responses  
● Sentiment – Positive, Neutral, Negative  
● Satisfaction Level – High, Medium, Low  
● Course Recommendation – Recommendation score (1–10)  
● Subject Knowledge – Rating  
● Concept Clarity – Rating  
● Presentation Quality – Rating  
● Student Support – Rating  
● Course Structure – Rating  
● Doubt Resolution – Rating  
● Assignment Difficulty – Rating  

Objectives of the Project  
● Analyze overall student satisfaction  
● Measure feedback volume and sentiment  
● Understand satisfaction levels across students  
● Analyze course recommendation trends  
● Evaluate different course quality attributes  
● Identify areas for improvement in teaching and support  

Dashboard Features  
● KPI Cards  
  ● Overall Rating  
  ● Total Feedback Count  
  ● Neutral Sentiment  
  ● Average Course Recommendation  

Visualizations Used  
● Bar Chart – Average Student Feedback Ratings  
● Column Chart – Average Rating by Satisfaction Level  
● Bar Chart – Course Recommendation by Satisfaction Level  
● Donut Chart – Student Satisfaction Distribution  
● Column Chart – Course Recommendation Score Distribution  

Filters (Slicers)  
Satisfaction Level  
Sentiment  
Course Recommendation Score  

Tools & Technologies Used  
● Microsoft Power BI  
● Power Query – Data cleaning & transformation  
● DAX – Measures and calculations  
● Microsoft Excel / CSV  
● GitHub – Version control & project hosting  

Data Cleaning & Preparation (Step-by-Step)  
● Imported survey dataset into Power BI  
● Removed duplicate and incomplete responses  
● Standardized rating scales  
● Converted numeric columns  
● Created calculated measures using DAX  
● Validated relationships and visual interactions  

Measures Created  
● Overall Rating  
● Total Feedback Count  
● Neutral Sentiment Count  
● Average Course Recommendation  

Key Insights :  
● Most students fall under the high satisfaction category  
● Overall course rating indicates positive student experience  
● High satisfaction students show higher recommendation scores  
● Course attributes such as subject knowledge and concept clarity score well  
● Low satisfaction levels highlight potential improvement areas  

How to Use This Project :  
Download the repository  
Open Student_Feedback_Analysis.pbix in Power BI Desktop  
Refresh the dataset if required  
Use slicers to filter satisfaction and sentiment levels  
Interact with visuals to explore insights
